# Projekt - Gry

Projekt zawierający gry: Kółko i Krzyżyk oraz Kamień, Papier, Nożyce.

## Nauczasz się:
- **Kółko i Krzyżyk:**
  - Tablice 2D - plansza gry
  - Pętle - inicjalizacja przycisków
  - Warunki logiczne - sprawdzanie zwycięzcy
  - Logika gry - turowanie między graczami
  
- **Kamień, Papier, Nożyce:**
  - Random - losowanie ruchu komputera
  - TextView - wyświetlanie wyników
  - Logika gry - porównywanie wyborów
  - Punktacja - śledzenie wyniku
  
- **Wspólne:**
  - Intent - nawigacja między grami
  - Button listeners - obsługa kliknięć
  - Zmienne instancyjne - przechowywanie stanu gry
  - Metody prywatne - organizacja kodu
